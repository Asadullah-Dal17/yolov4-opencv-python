## Installation

You Need [**Opencv Contrib**](https://pypi.org/project/opencv-contrib-python/)

--> **windows**
```
pip install opencv-contrib-python==4.5.3.56
```
--> **Linux or Mac**

```
pip3 install opencv-contrib-python==4.5.3.56

```
